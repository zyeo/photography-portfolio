# Plan 02-03 — Build first image ingestion path

## Goal
Create the first admin upload path that extracts EXIF where available and persists photo records without requiring metadata completeness.

## Tasks
1. Add EXIF normalization helpers.
2. Add server route for authenticated uploads into private originals storage.
3. Add minimal admin upload UI and response handling.

## Must Haves
- Upload creates a photo record.
- EXIF values populate when present and remain nullable when absent.
- Hero focal point and mobile crop fields exist in persisted shape.
