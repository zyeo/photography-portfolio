# Plan 02-01 — Create data and storage spine

## Goal
Define the relational schema, storage buckets, policies, and local setup guidance that every later workflow depends on.

## Tasks
1. Add SQL migration for photos, journal entries, collections, and photo collections.

3. Add seed data and local setup documentation.

- Seed data demonstrates nullable metadata and core relationships.
