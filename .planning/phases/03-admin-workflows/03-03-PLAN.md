# Plan 03-03 — Build curation workspaces

## Goal
Add the visual curation spaces that turn the photo library into a public-facing portfolio system.

## Tasks
1. Add Selected workspace with ordering and size controls.
2. Add Collections management and photo assignment basics.
3. Add Homepage hero workspace with pin/unpin and crop controls.

## Must Haves
- Selected, collections, and homepage heroes can be curated without duplicating photo records.
- Removing from Selected preserves the underlying photo.
- Homepage allows one pinned hero or rotation-ready approved photos.
